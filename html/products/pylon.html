매니지드 보안 서비스 파일론

중소기업의 통합 네트워크 보안을 위한 매니지드 보안 서비스

  • 제품소개
  • 특장점
  • 주요기능

All in PYLON 통합 네트워크 보안, 파일론

파일론(PYLON)은 소호 및 중소기업이 가진 한정된 자원 속에서도 비인가 트래픽 차단, 네트워크 통제, 해킹 등의 침입 탐지 등 네트워크를 노리는
포괄적인 보안 위협에 대응할 수 있도록 통합 네트워크 보안을 위한 매니지드 보안 서비스(Managed Security Service*)를 제공합니다. 초소형 보안 센서와의 간단한 연결만으로
파일론에서 제공하는 매니지드 보안 운영 센터를 통해 중앙에서 침해 위협을 파악하고 대응을 가능케해 외부 네트워크 공격으로부터의 내부 자산을 안전하게 보호합니다.

*Managed Security Service(MSS) : 전문 보안인력 및 보안 운영 센터를 보유한 서비스 제공자에 의해
아웃소싱되는 네트워크 위협 관리, 침입탐지, 안티바이러스, 취약성 테스트 등을 포함하는 네트워크 보안 서비스

파일론이 특별한 이유

비용, 인력, 물리적 공간 등의 제약으로 보안 시스템 도입에 부담을 느낄 수밖에 없는 중소기업을 위해 파일론이 통합 네트워크 보안 서비스를 제공합니다. 초소형 보안센서를
연결하면 즉시 클라우드 서비스에 접속하여 인터넷과 연결된 우리 회사의 네트워크 보안에 대한 가시적 관리가 가능합니다. 홈네트워크를 이용하는 개인에서부터 소규모 사업장을
운영하는 소호·중소기업까지 필수적인 네트워크 보안과 보안 전문가의 매니지드 보안 서비스를 기존 네트워크 통합 보안 시스템의 1/10 가격으로 누리실 수 있습니다.

  • 초소형 통합 보안 센서
    7.5cm x 6cm x 3.5cm의 초소형
    보안 센서로 통합 네트워크 보안
  • 쉬운 설치 및 운영 관리
    Plug & Play 방식의
    손쉬운 설치 및 관리
  • 중소기업 도입 부담 제거
    상용 서비스의 1/10
    도입 부담 줄인 월과금 정책
  • 보안 전문가의 관리형 보안 서비스
    보안/운영/모니터링
    매니지드 보안 서비스 제공

파일론으로 누리는 네트워크 보안

강력한 보안을 제공하는 초소형 보안 센서를 공유기에 연결하는 것만으로 파일론의 네트워크 보안이 시작됩니다. 오피스/공장/전국 지점 등과 연결된 POS 기기, 키오스크, ATM기 등
임베디드 시스템의 해킹 방지부터 안전한 연결을 위한 홈 IoT 게이트웨이 보안까지 책임집니다. 전문 인력 없어도 바로 시작하는 중소기업 네트워크 보안, 지금 경험해 보시기 바랍니다.
