스마트 협업의 시작, 오피스하드 VEX

오피스하드 VEX(OfficeHARD VEX)는 디지털 자산의 저장과 공유, 협업, 자료 보안, 정보 통합 등의 효율적인 기능을 제공하는
국내 보안 서버 점유율 1위 기업용 보안 파일 서버입니다. 부서와 업체 간의 정보 공유와 공동 작업을 위한 다양한 기능과 함께 스마트한 협업 환경을 제공합니다.
랜섬웨어나 악성코드와 같은 보안 위협으로부터 기업 중요 자산을 보호하고 정보 유출에 대한 선제적 대응도 탁월한 보안 파일 서버를 지금 만나보시기 바랍니다.

오피스하드 VEX가 특별한 이유

오피스하드 VEX는 다양한 운영 정책으로 기업의 정책별 요구 사항을 반영하며 최적화된 보안 업무 환경을 제공하고 있습니다.
오피스하드 VEX를 통해 기업 내부 정보 보안을 더욱 강화시키고 국가기관의 보안 권고와 지침 반영에도 걱정 없이 대응할 수 있습니다.
랜섬웨어와 정보 유출 사고로부터 기업의 중요 정보를 안전하게 보호할 수 있는 방법, 오피스하드 VEX가 알려드립니다.

  • 악성 위협 선제 대응
    랜섬웨어와 악성 위협으로부터
    기업의 중요 자산 보호
  • 내부 정보 보안 강화
    효과적인 권한 설정
    암호화 및 접근 통제 강화
  • 효율적 협업 방식
    협업을 위한 팀/부서 폴더, 개인폴더
    공유함, 파일 송/수신함 제공
  • 사용 편의성 우수
    멀티 웹 브라우저 환경 지원,
    실시간 통계 모니터링

오피스하드 VEX로 누리는 안전한 협업 생활

오피스하드 VEX가 제공하는 웹 탐색기에서의 직접 편집은 서버에 암호화된 형태로 저장되어 랜섬웨어에 감염된 파일이 업로드되어도
서버 내 다른 문서에게 추가 감염을 발생시키지 않는 가장 큰 장점이 있습니다. 또한 부서 간 정보를 공유 시 효율적인 협업을 위한
개인폴더 공유함, 파일 송/수신함을 제공하며 실수로 인한 정보 삭제에도 즉시 복구가 가능하도록 설정할 수 있어 안전한 협업이 가능해집니다.

  • 전용 웹 탐색기 지원
    전용 웹 탐색기 내 열람으로 랜섬웨어
    확산 방지 탐색기 내 직접 편집 가능
  • 검색 최적화 기능
    본문 추출 엔진이 기본 탑재
    빠르고 편리한 본문 검색 지원
  • 권한 관리
    폴더별 사용자 및 그룹 세부 권한
    다수 시스템 중간 관리자 설정
  • 백업 설정 기능
    사용자 기준 별 백업 파일 및
    스케줄 설정 및 문서 유실 방지
  • 파일 보안
    파일 전송구간 암호화 지원
    파일 저장 시 보안폴더 암호화
  • DB 보안
    IP 별 접근 제약 및 접속 정보 및
    사용자 개인 정보 암호화 및 SSL 지원
  • 로그관리 및 사용 통계
    파일 이용 현황 일/월별 리포트
    모든 사용 내용 로그 지원
  • 다양한 부가 기능
    시스템 리소스 모니터링
    다양한 오픈 API 제공

오피스하드 VEX 도입 기업들
